What is Css

  1. Css stands for cascading style sheet
  2. css describes how html elements are to be displayed on screen,paper or in other media
  3. css is not necessary saves a lot of work. it can control the layout of multiple web pages all at once
  4. External stylesheets are stored in CSS files

Css can be added to HTML document in 3 ways :

  • Inline -by using the style attribute inside HTML elements 1
  • internal by using the style element in the head section
  • External - by using a link element to link an external css file
  • The most common way to add CSS , is to keep the css style in external CSS files.HOWEVER ,in this tutorial we will use Inline and internal styles, because this is easier to demonstrate , and easier for you to try it yourself.

When comparing CSS class vs ID, the difference is that CSS class applies a style to multiple elements. ID, on the other hand, applies a style to one unique element. ID is also
special in that you can use a special URL to link directly to an element and it's used by JavaScript.

In Css, selectors are used to target a specific element or range of elements on a web page. Once an element has been targeted, a style or set of styles can be applied to the
element.

There is a wide range of selectors available. Two of the most commnly used are class and ID. Both are used to elements to which a style should be applied.
